ps

  2. 学习用品中的书立和收纳箱可以迟点再买，主要看教室位置够不够放收纳箱，够就买收纳箱放座位旁边地上，不够就买书立放桌子上

  4. 彩色笔推荐购买与中性笔类似的那种，要握感舒适的，主要是做笔记(三四种颜色最佳)和错题改正(红笔)

Q&A

日常生活(内宿)

Q：平时生活费费一般算多少
A：伙食费按每顿8元来记，每个月约为8×3×30=720；加上杂七杂八的一个月大约900银子

关于服装

Q：有几套校服？
A：夏装三套冬装三套无礼服

Q：校服怎么量尺码的？
A：开学时会量身高的

Q：校服费是多少？
A：约600

Q：每天都要穿校服吗？
A：学校规定要穿

Q：军训服装是怎样的？
A：上衣迷彩短袖，裤子纯黑短裤，帽子外迷彩内原谅绿，无统一配鞋

关于宿舍

Q：一间宿舍多少人？
A：女生宿舍B栋12个床位，男生宿舍新栋(腾业楼隔壁)及A栋(饭堂隔壁)12个床位

Q：床的大小
A：大概在0.9×1.9m左右

Q：宿舍有没有插头？
A：想都别想！！！

Q：宿舍配置
A：一冲凉房一卫生间一小阳台，宿舍有窗没帘

Q：能不能挂蚊帐、床帘？
A：蚊帐是可以的，至于床帘嘛…不确定

Q：有没有空调？风扇多少个？
A：没有空调，两个风扇

关于教室

Q：教室在哪？
A：到时候会有学姐带；AB级一个在图书馆另一个在腾业楼，C级在文星楼

Q：教室多大？
A：没量过，图书馆最大(剩余空间最大)，其次文星楼(略有富余)，最小腾业楼(拥挤)

Q：有没有空调？几个空调？
A：每班两台空调，但是腾业楼那边因为教室太多限流只能开一台，电用完就停，图书馆和文星楼都是两台都能开

关于报到

Q：报到要带的东西
A：

钱生活用品手机
外宿700
内宿1500-2000
校服费生活费
约600元每月700-900
教辅费(练习册)印刷费
约100约80

Q：报到时间是早上还是下午？
A：按往届的情况是早上和下午都可以的，不过还是尽量早上吧，下午都没啥人

Q：报到后要干嘛？
A：外宿去浪，内宿整理完自己的东西去浪

Q：报到那天要上晚修吗？
A：看具体安排，但是这么多届都是要的

Q：报到那天家长能进去吗？
A：往届是可以的，只是…尽量别进去吧

Q：可以家长开车送进去吗？
A：可以开车送到校门口，但是不能进去；另外，报到当天可能会特别挤，可以让家长把车停到燕城地下停车场或送到城中新街与潮流城的路口以免拥堵

Q：有没有学长学姐迎接？
A：往届都会有，只是…这回恰逢疫情…咱也不知道

Q：报到时间是上午还是…
A：往届都是一整天都可以，只是下午人很少，只要别晚上到就行(晚上要上晚修)

Q：报到完要干嘛？
A：外宿的签完名交完钱就可以回家或者去浪了，内宿的整理完自己的东西就可以去浪

关于学习

Q：中考不小心考差了还能不能进实验级奥赛班？
A：一中每年都很通融，差那么一两分还是给的(然后就造成了我们这届实验级单一中302人)，分差得太多也不要灰心，新高考选科时(高一下学期)还会再分一次班，到时实验级估计也会压缩人数

Q：想买教辅在哪买比较好？
A：一中附近有广轩和广明这两家比较大的书店(高中教辅打7折)，也可以到步行街的文华书城(打折情况不明，我去是有的)

Q：晚修是干嘛的？
A：完成老师布置的作业

Q：晚修和晚练有什么区别？
A：本质上没有啥区别，只是在某些班级(如实验级)会安排来做限时训练

Q：晚修各科有没有具体的安排？
A：实验级是有安排的，科任老师也会根据时间来安排具体任务

Q：有没有美术课、音乐课、电脑课等？
A：有音乐课、美术课，每两周一节；心理课每周一节；没有电脑课

关于分科

Q：选科的'3+1+2'是怎样的
A：高中的主科为语文、数学、外语、物理、历史、化学、生物、政治、地理，关于'3+1+2'的详细信息如下

选科规律如下：

  • “3”为必选科目
  • “1”为二选一
  • “2”为四选二
  • “外语”又分英语和日语
  • 每个考生必选“语数”三科

关于课外生活

Q：有哪些课外活动
A：一中有四大组织(学生会、团委、志愿者、广播站)；一中还有社团和管理社团的社联，四大组织将于18-21日开展招新

Q：这些能同时参加几个
A：一般来说四大组织和社联只能去一个，但可以参加多个社团(只要你忙得过来)

Q：四大组织是干嘛的
A：点击查看关于学生会的介绍，有任何疑问或想了解其他组织的信息请在新生群@学长学姐
